# FarlandTacticsActionRPG

환경 : C++, Visual studio2017, FMOD, GDI+

주요 구현 사항
  - Camera Based Render
  - Behavior Tree AI
  - Factory Pattern
  - A* Path Finding Algorism
  - Flood Fill Algorism
  - Z-order(Y-sort)
  - Render Clipping(Map Tile Culling)
  - Player, NPC, Enemy, Boss
  - Player Skill(Range, Warning UI, Action Time)
